***********************ВАЖНО**************************
1! Все блоки написаны для использования в CFC схемах
2! Тестировались на CPU: 315PN/DP, 317PN/DP

**********************ОПИСАНИЕ************************
0_PUT_GET - межпроцессорное взаимодействие между двумя
ПЛК через DATA BLOCK.

1_ONOFF_D - задержка отключения контактора, при отклю-
чении насоса ИНС, сначала отключается ПЧ, затем отклю-
чается контактор.
 
2_PID_INS - ПИД регулятор насоса ИНС.

3_F_ANALYSYS - анализ возможного отказа насоса на ИНС,
останов отказавшего насоса и переключение на рез-ный.
Отказ определяется при падении расхода через датчик
до нуля при включеном двигателе, для фильтрации ложных
срабатываний и переключений, введена задержка t сек.

4_GATE_RS - триггер переключения ручного и автоматичес-
кого режимов работы затвора, для правильной фиксации и
сброса текущего режима работы затвора.

5_FILLING - Цикл заполнения илоуплотнителей по времени,
используется относительное время с кратностью 10 сек.

6_EMPTYING - Цикл слива илоуплотнителей по состоянию
линейки датчиков уровня НС уплотненого ила, активен в
фазе ожидания наполнения илоуплотнителей, работае задан-
ное количество раз на каждом из затворов слива.
